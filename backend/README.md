# Backend

## Quick Start

1. Create a virtual environment.
2. Install dependencies with `pip install -r requirements.txt`.
3. Copy `.env.example` to `.env` and update the values.
4. Start the server with `uvicorn app.main:app --reload`.

## Current Scope

- FastAPI application skeleton
- API route placeholders for the four major business domains
- Service layer stubs returning mock data for front-end integration
- Alert agent and websocket manager scaffolding
