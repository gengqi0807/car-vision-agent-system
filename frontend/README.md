# Frontend

## Quick Start

1. Install dependencies with `npm install`.
2. Start the dev server with `npm run dev`.
3. Ensure the backend runs on `http://127.0.0.1:8000`.

## Current Scope

- Vue 3 + TypeScript + Vite shell
- Route structure for dashboard, plate recognition, police gesture, owner gesture, and alert monitoring
- Shared placeholder components for video, overlay, control panel, and alert list
- Mock-friendly API layer ready for real backend endpoints
